# DATA ANALYSIS ON SUPERSTORE TRANSACTION USING EXCEL
## Introduction:
    This Project is on sale analysis of super tranasctions (with 31223 rows of data) using excel formula, power query and power pivot

## Sales Dashboard:
    1. This Dashboard will provide the "Sales_Median", "Sales_Count" and "Profit_Average" based on selection of "Category", "Country"and "Year".
    2. Graph represents the Sales Average with category, Sales Average with Country and Sales Average with Year
    3. Excel Formula and Charts are used to acheive this dashboard.

## Power Query:
    1. Load and Transform Data of orders table
    2. In Sales table, calculated the duration by adding Custom
    3. Created a new table named Calender by Inserting Month, Year and Quarter Column and removing the other columns and duplicates

## Power Pivot:
    Relationship between tables were created and DAX is used for calculation to create interactive pivot table and pivot chart with slicers.
        1. Average Order Value based on Sub-Category with Market as slicer
        2. Region wise Revenue Average, Revenue Median and Total Orders with sub-category as slicer
        3. Quarter wise Revenue Average with sub-category as slicer
