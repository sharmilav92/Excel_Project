# DATA ANALYSIS ON SUPERSTORE TRANSACTION USING EXCEL
## Introduction:
    This Project is on sale analysis of super tranasctions (with 31223 rows of data) using excel formula, power query and power pivot

## Sales Dashboard:
    1. This Dashboard will provide the "Sales_Median", "Sales_Count" and "Profit_Average" based on selection of "Category", "Country"and "Year".
    2. Graph represents the Sales Average with category, Sales Average with Country and Sales Average with Year
    3. Excel Formula and Charts are used to acheive this dashboard.
